# Початок роботи - Інструмент покращення аудіо REAPER

## Короткий огляд

Цей інструмент допомагає покращити якість аудіо:
- **Виявлення та зменшення фонового шуму** з вашого аудіо
- **Аналіз відео** для розуміння контексту сцени
- **Пропозиція оточуючих звуків** які відповідають кожній сцені
- **Створення доріжок REAPER** з усіма пропозиціями готовими до використання

## Покроковий посібник

### 1. Запуск програми
```bash
bash launch.sh demo
```
Або використовуйте графічний запускач, якщо доступний.

### 2. Виберіть свої файли

**Аудіо файл (Обов'язково):**
- Натисніть "Вибрати аудіо"
- Виберіть свій аудіо файл (WAV, MP3, FLAC, тощо)

**Відео файл (Опціонально, але рекомендується):**
- Натисніть "Вибрати відео"
- Виберіть свій відео файл (MP4, MOV, AVI, тощо)
- Програма буде аналізувати сцени у вашому відео

### 3. Аналізувати
- Натисніть **"1. Аналізувати"**
- Дочекайтеся завершення аналізу
- Програма буде:
  - Виявляти фоновий шум
  - Аналізувати сцени відео (якщо надано)
  - Генерувати пропозиції аудіо для кожної сцени

### 4. Обробити аудіо (Опціонально)
- Відрегулюйте повзунок **"Сила зменшення шуму"** (0.0 = немає, 1.0 = максимум)
- Натисніть **"2. Обробити аудіо"** щоб зберегти аудіо з зменшеним шумом
- Це створює окремий аудіо файл, який ви можете використовувати де завгодно

### 5. Експортувати в REAPER
- Натисніть **"3. Експортувати в REAPER"**
- Програма буде:
  - Генерувати файл проекту REAPER
  - Створювати аудіо доріжки з пропозиціями
  - Встановлювати скрипт імпорту відео
  - Показувати вам наступні кроки

### 6. Імпортувати в REAPER
- Натисніть **"4. Імпортувати в REAPER"**
- REAPER відкриється з вашим проектом
- Ви побачите повідомлення про успіх з інструкціями

### 7. Додати відео (Якщо ви вибрали відео файл)

**Варіант A: Автоматичний (Рекомендується)**
1. У REAPER: **Actions > Show action list**
2. Натисніть **"Load ReaScript..."**
3. Перейдіть до: `~/Library/Application Support/REAPER/Scripts/add_video_to_reaper.lua`
4. Натисніть **Open**
5. Натисніть **Run**
6. Доріжка відео буде створена автоматично!

**Варіант B: Вручну**
1. У REAPER: **Insert > Media File**
2. Виберіть свій відео файл
3. Натисніть **View > Video Window** (Cmd+Shift+V)
4. Натисніть **Play**

### 8. Переглянути свій проект

- **Вікно відео**: View > Video Window (Cmd+Shift+V)
- **Аудіо доріжки**: Ваше оригінальне аудіо + запропоновані доріжки покращення
- **Відтворення**: Натисніть Play щоб почути своє аудіо з відео

## Параметри

### Мова
- Змініть між англійською та українською мовами
- Впливає на весь текст інтерфейсу та повідомлення

### Параметри REAPER OSC
- Налаштуйте з'єднання з REAPER (розширене)
- За замовчуванням: 127.0.0.1:9000

### VLC Плеєр
- Автоматично виявляється та налаштовується
- Потрібен для відтворення відео в REAPER
- Завантажити: https://www.videolan.org/vlc/

## Поради та трюки

### Найкращі результати
- Використовуйте чисті аудіо файли (WAV або FLAC рекомендуються)
- Використовуйте відео файли, які відповідають тривалості вашого аудіо
- Відрегулюйте силу зменшення шуму за своїм бажанням

### Зменшення шуму
- **0.0**: Без зменшення шуму (оригінальне аудіо)
- **0.5**: Помірне зменшення шуму (рекомендується)
- **1.0**: Максимальне зменшення шуму (може видалити деякі деталі аудіо)

### Кілька пропозицій
- Програма пропонує до 13 різних оточуючих звуків
- Кожна пропозиція знаходиться на окремій доріжці REAPER
- Вимкніть або відрегулюйте гучність будь-якої доріжки, яку ви не хочете

### Експорт вашої роботи
- Використовуйте функції експорту REAPER для збереження вашого остаточного мікшу
- Усі доріжки готові до подальшого редагування

## Усунення неполадок

### Відео не показується
- Переконайтеся, що VLC встановлено (Параметри показують статус)
- Закрийте та перезавантажте REAPER після додавання відео
- Перевірте, що шлях до відео файлу правильний

### Аудіо не відтворюється
- Перевірте, що аудіо файл дійсний (спробуйте в іншому плеєрі)
- Перевірте параметри аудіо REAPER
- Переконайтеся, що аудіо пристрій підключено

### Скрипт не завантажується
- Перевірте, що файл скрипту існує: `~/Library/Application Support/REAPER/Scripts/add_video_to_reaper.lua`
- Спробуйте перезавантажити REAPER
- Перевірте, що REAPER має дозвіл на доступ до файлу

## Розташування файлів

**Приклади файлів:**
```
~/Projects/ihor-audio-main/reaper-audio-enhancement/assets/sample_files/
  - sample_video.mp4 (4K відео снігопаду)
  - sample_audio.wav (видобуте аудіо)
```

**Проекти REAPER:**
```
~/.reaper_audio_enhancement/exports/
  - project_YYYYMMDD_HHMMSS.rpp
  - export_YYYYMMDD_HHMMSS.json
```

**Скрипт відео:**
```
~/Library/Application Support/REAPER/Scripts/
  - add_video_to_reaper.lua
```

## Резюме

Інструмент розроблений для інтуїтивного використання:
1. **Виберіть файли** → 2. **Аналізуйте** → 3. **Обробляйте** → 4. **Експортуйте** → 5. **Імпортуйте** → 6. **Додайте відео** → 7. **Насолоджуйтеся!**

Це все! Ваше аудіо тепер покращено з пропозиціями, які відповідають сценам, та відео.
